# KIW - Technical Documentation

KIW is a sophisticated B2B e-commerce platform designed for efficiency and scalability. It features a modern single-page application (SPA) architecture, providing a seamless experience for administrators, sellers, and customers.

## Technology Stack

- **Backend**: PHP 8.2 / Laravel 10
- **Frontend**: Vue.js 3 / Vite / Vuetify 3
- **Database**: MySQL & Redis
- **Auth**: Laravel Passport (OAuth2)
